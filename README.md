# Iris Flower Prediction (Streamlit ML App) 
A beginner-friendly Machine Learning project built with: - Python - Scikit-learn (KNN model) - Streamlit (web app) 
## Features - Takes 4 Iris measurements - Predicts Iris flower type: - Setosa - Versicolor - Virginica - Shows confidence score
